## Overview
Using the seeed studio xiao esp32-s3 Sense SoC to learn embedded programming. 
I'm, writing in C and using [esp-idf](https://idf.espressif.com/) to build and flash to the board.

## Current Features
- Onboard camera takes in video
- Connects to a local Wi-Fi network
- Hosts an http server which serves the video stream over the local area network
- Viewable in web browser

## Future Plans
- Pair 2 boards, 1 facing out and 1 facing in, mounted on a pair of glasses or frame
- Use an ML model with image embeddings for eye tracking and image recognition
